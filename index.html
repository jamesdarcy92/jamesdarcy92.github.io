About me

Hello! I'm James Darcy, a seasoned software developer with a decade of expertise in Full Stack Development.

- TYP: Implemented Learning2Write which is an SMS Android application whose main feature was to prevent users from sending texts when inebriated.

Overview:
During the course of my degree in Dublin City University, I gained valuable experience in a number of areas of Computer Science and Software Development.
The two main projects I implemented during this time were:
Learning2Write an Android application whose purpose is to assist in the process of teaching young children to write letters of the alphabet. The app rendered a number of points on screen, which make up the shape of a letter. Four versions of the letter are displayed with varying amounts of points. Each point detects if a touch intersects it and if all are traversed through, the letter is considered as being completed successfully. Learning2Write allows for separate student and teacher logins, where the teacher's view will allow for them to keep track of what letters the students have drawn, how many times they have attempted a certain letter, and how many times they drawn the letter successfully.
extendText an SMS Android application whose main feature was to prevent users from sending texts when inebriated. Users could enable the feature and before sending a text would be required to pass a number of sobriety tests which included memory, reactionary and arithmetic tests.

Throughout my degree, I was taught an array of modules such as: Operating Systems, Object Oriented Design and Models, Data Structures & Algorithm Design, Mathematics, Software Patterns & Design, Computability & Complexity, Computer Networks, Web Design, Systems Analysis, Advanced Algorithms & A.I Search, Human Computer Interaction, Software Testing, Multimedia Technology, Compiler Construction, Computer Graphics and, Cryptography & Security Protocols.
